1. Цель работы

Познакомиться со структурой HTML-документа, создать страничку со своей презентацией, проявить творческий подход.

2. Задание 1. HTML Academy

Были пройдены все бесплатные курсы на HTML Academy.

3. Задание 2. Сайт о себе

Сайт представляет собой 3 страницы: главная страница, которая содержит контакты для связи со мной, страница "Обо мне", где можно увидеть краткую сводку моего жизненного пути, а так же страницу с отчетом.

4. Задание 3. Отчет на HTML

Был написан отчет на HTML с сохранением всех атрибутов.

5. Вывод

В результате лабораторной работы познакомился с HTML, CSS, создал сайт о себе.
